💰 Compound Interest Microservice

Welcome to the Compound Interest Calculator API! This service calculates compound interest based on the provided parameters.

📋 API Endpoint

GET/api/compound-interest

Query Parameters:

  • p (required): Principal amount (initial investment)
  • r (required): Annual interest rate (as a percentage, e.g., 5 for 5%)
  • y (required): Number of years

Example Requests:

Calculate compound interest on $1000 at 5% annual rate for 2 years:

GET /api/compound-interest?p=1000&r=5&y=2

🧮 How It Works

The API uses the compound interest formula:

A = P(1 + r/100)y

Where:

  • A = Final amount (principal + interest)
  • P = Principal amount
  • r = Annual interest rate (percentage)
  • y = Number of years
